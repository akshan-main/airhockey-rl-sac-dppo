# airhockey-sac-v0

5,000 demonstration episodes for a custom 2D air hockey environment,
generated by a Soft Actor-Critic agent trained via self-play for 1M
environment steps.

## Environment

`AirHockey2D-v0` — a 320×540 top-down 2D air hockey rink with two
paddles. Continuous 2D action space (paddle accel x, accel y),
10-D fully-observed observation, ±10 sparse goal reward + 0.1
puck-contact bonus + 0.01 shaping toward opponent goal per step.

Source: [github.com/akshan-main/airhockey-rl](https://github.com/akshan-main/airhockey-rl)

## How it was generated

```bash
python -m airhockey.train_sac --total-steps 1000000 --out ckpt/sac_expert.pt
python -m airhockey.collect --expert ckpt/sac_expert.pt --episodes 5000
python scripts/build_minari_dataset.py --id airhockey-sac-v0
minari upload airhockey-sac-v0
```

The collector runs two copies of the trained SAC agent against each
other and records the bottom paddle's perspective at every step.

## Intended use

Behavior cloning and offline-RL pretraining for diffusion policies,
action chunking transformers, or any other method that learns from
(state, action) pairs. Used directly as the Stage 1 dataset for the
[`akshan-main/airhockey-dppo`](https://huggingface.co/akshan-main/airhockey-dppo)
Diffusion Policy.
